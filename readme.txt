这是一个用来斗鱼弹幕自动点歌的后台，由于推流的未知原因，推流最终还是只能依赖使用OBS
因此这个后台实际上只是在能根据房间里面的弹幕自动选择下一首播放的歌曲，同时会生成一个反应
相关信息的图片，有OBS将音效和图片推送到斗鱼从而实现弹幕点歌的功能

目前支持的功能有，自动从QQ音乐搜索下载点的歌曲，由于空间限制，只能下载第一首，各位点歌的时候可以注意
弹幕点歌的操作格式为mc:歌名
后台即可自动实现点歌，当没有歌曲在列表中时，会循环播放hush（我妹子喜欢的音乐）

实现分为三个部分

获取弹幕部分
get_barrage.py
需要在命令行输入房间号，推荐房间号3200597（我的弹幕），即可将该房间内的弹幕通过斗鱼的API下载到本地

点歌处理部分
get_stream.py
根据弹幕，选歌，生成图片

图片显示部分
show_image.py
本来可以集成到上一部分，但是一直有bug，只能单独作为一个模块

依赖的库为socket, requests, PIL, cv2，本地需要有mplayer(播放音乐)
命令为
python3 get_barrage.py {你的房间号} | python3 get_stream.py | python3 show_image.py > log.txt

最后在本地用OBS推流即可
